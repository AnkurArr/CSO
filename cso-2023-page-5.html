25) Which device is used to point and select objects on the screen?

✔ Mouse

A mouse controls the pointer and is used for selecting, dragging and clicking.

26) Which number system uses digits 0–9 and A–F?

✔ Hexadecimal

Hexadecimal has base 16 and uses 16 symbols (0–9 and A–F).

27) The device that converts digital data into sound is —

✔ Speaker

Speakers convert processed digital signals into audible sound.

28) ROM stands for —

✔ Read Only Memory

ROM contains permanently stored instructions — cannot be erased easily.

29) Which of the following is temporary storage?

✔ RAM

RAM stores data only while the computer is ON — it is volatile memory.

30) The smallest unit of data in a computer is —

✔ Bit

A bit represents a single binary value: either 0 or 1.
