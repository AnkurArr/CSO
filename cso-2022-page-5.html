29) ______ is used to convert high level language into machine language.

✔ Compiler

Compiler পুরো প্রোগ্রাম translate করে binary বানায়।

30) ______ holds the data temporarily during processing.

✔ Main memory (RAM)

RAM volatile — power off হলে data থাকে না।

31) ______ is the brain of the computer.

✔ CPU

CPU সব control + calculation করে।

32) ______ controls the flow of data & instructions.

✔ Control Unit (CU)

CU signals দিয়ে পুরো system পরিচালনা করে।

33) ______ converts digital signals to analog signals.

✔ DAC (Digital to Analog Converter)

DAC → digital → analog waveform এ পরিবর্তন।

34) ______ is used to convert analog signal to digital.

✔ ADC (Analog to Digital Converter)

ADC → analog input কে digital bits এ দেয়।
