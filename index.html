Computer System Organization — PYQ (2022) — Page 1

1) The number successful accesses to memory stated as a fraction is called —

✔ Hit rate

Hit rate = cache hits / total memory access.

2) The final addition sum of 0110 & 0110 is —

✔ 1100

6 + 6 = 12 → binary 1100.

3) CSA stands for —

✔ Carry Save Addition

Used in fast adders to reduce carry delay.

4) Individual control word of micro routine is called —

✔ Micro instruction

Each micro-step = one micro-instruction.

5) Circuit that converts binary to decimal —

✔ Decoder

n-bit input → 2ⁿ unique outputs.

6) Situation when data for operation is not available —

✔ Data hazard

Occurs in pipeline when next instruction needs unfinished data.

7) Full form of CISC —

✔ Complex Instruction Set Computer

Supports many complex instructions.

8) Alternate of ADD #5,R1 is —

✔ ADDI 5,R1

#5 = immediate constant.

9) To read multiple bytes at same time we use —

✔ Cache

Cache fetches blocks, not single bytes only.

10) Full adder sum circuit uses —

✔ XOR

Sum = A ⊕ B ⊕ Cin.
