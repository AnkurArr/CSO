Answer: Carry Save Addition

Answer: Micro-instruction

Answer: Decoder

Answer: Data hazard

Answer: Complex Instruction Set Computer

Answer: ADDI 5, R1

Answer: Cache

Answer: XOR

Answer: IN, OUT

Answer: Move
