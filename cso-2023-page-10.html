55) Which unit manages all input and output operations?

✔ I/O Unit

The Input/Output unit controls communication between the CPU and external devices.

56) Binary number 101 equals —

✔ 5 (decimal)

1×4 + 0×2 + 1×1 = 5.

57) Which device is used to store backup data?

✔ External storage (like USB / external HDD)

Backups are usually stored on removable secondary storage devices.

58) The unit that coordinates all parts of the computer is —

✔ Control Unit

The Control Unit sends control signals so every part works in correct order.

59) The binary equivalent of decimal number 4 is —

✔ 100

4 = 1×4 + 0×2 + 0×1 → binary 100.

60) Which of the following is NOT an output device?

✔ Keyboard

Keyboard is used to enter data, so it is an input device — not output.
