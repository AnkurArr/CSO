11) Computer address bus is —

✔ Unidirectional

CPU → Memory/IO only.

12) Bus connecting CPU to memory on board —

✔ System bus

Address + Data + Control.

13) Instructions for I/O —

✔ IN, OUT

IN → read port, OUT → write port.

14) R1 ← R2 indicates —

✔ Register transfer

Copy contents of R2 into R1.

15) Interrupt that can be ignored —

✔ Maskable interrupt

Software can disable it.

16) Permanent storage for data/instructions —

✔ Secondary storage

Hard disk / SSD stores permanently.
