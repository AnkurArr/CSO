17) ______ is used to store data, instructions and results permanently.

✔ Secondary storage (Hard disk / SSD)

Permanent storage = non-volatile memory.

18) ______ is generally used to increase the apparent size of physical memory.

✔ Virtual memory

Secondary storage behaves like extra RAM.

19) Gray code is also called ______.

✔ Unit distance code

Only one bit changes at a time → error কম হয়।

20) Instruction register stores ______.

✔ Currently executing instruction

IR = CPU যে instruction execute করছে।

21) High-speed memory between CPU & main memory is called ______.

✔ Cache

Cache access = খুব fast → CPU অপেক্ষা করে না।

22) I/O address in 8086 is ______ bit.

✔ 16-bit

8086 → 64K I/O ports (2¹⁶).
