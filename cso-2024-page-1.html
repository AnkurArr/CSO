1) Which memory is the fastest?

✔ Cache memory

Cache is closest to the CPU and has very low access time.

2) The brain of the computer is —

✔ CPU

The CPU controls and processes all instructions.

3) Binary number 10 equals —

✔ 2 (decimal)

Binary 10 means one 2 and zero 1’s → 2.

4) Which unit performs all calculations?

✔ ALU

ALU performs arithmetic and logical operations.

5) Which device is used to store large amount of data permanently?

✔ Hard disk / SSD

Secondary storage keeps data even when power is off.

6) ROM is —

✔ Non-volatile memory

Data remains stored even after power is switched off.
