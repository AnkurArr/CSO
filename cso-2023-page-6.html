31) The device that displays visual output is —

✔ Monitor

A monitor shows processed information as text, images and video.

32) Which component performs calculations?

✔ ALU

The Arithmetic Logic Unit performs arithmetic and logical calculations.

33) Which of the following is a primary memory?

✔ RAM

Primary memory is directly accessible by CPU — RAM is primary memory.

34) Decimal number 8 in binary is —

✔ 1000

8 = 1×8 + 0×4 + 0×2 + 0×1 → binary = 1000.

35) Which bus carries control signals?

✔ Control bus

The control bus sends timing and control signals to coordinate operations.

36) The full form of ALU is —

✔ Arithmetic Logic Unit

ALU is the part of CPU responsible for arithmetic and logical operations.
