73) Which device is used to display video output?

✔ Monitor

The monitor displays text, images and videos produced by the computer.

74) Binary number 0 equals —

✔ 0 (decimal)

Zero is the same in both decimal and binary.

75) Which component connects CPU with memory?

✔ System bus

System bus = address bus + data bus + control bus.

76) Which device shows hard copy output?

✔ Printer

Printer produces printed (paper) output.

77) The binary equivalent of decimal number 11 is —

✔ 1011

11 = 8 + 2 + 1 → binary = 1011.

78) Which part of the CPU performs operations?

✔ ALU

ALU performs arithmetic and logical operations during processing.
