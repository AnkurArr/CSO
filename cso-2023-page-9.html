49) Which number system is used inside the computer?

✔ Binary

Computers work with 0s and 1s because digital circuits have two states (ON/OFF).

50) The device used to print output on paper is —

✔ Printer

A printer produces hard-copy output from digital documents.

51) The full form of BIOS is —

✔ Basic Input Output System

BIOS starts the hardware and loads the operating system during boot.

52) Which memory is also called auxiliary memory?

✔ Secondary memory

Secondary memory (hard disk/SSD) is used for long-term data storage.

53) Decimal number 6 in binary is —

✔ 110

6 = 4 + 2 → binary representation is 110.

54) Which component interprets and executes instructions?

✔ CPU

The CPU fetches, decodes and executes instructions — the “brain” of the system.
