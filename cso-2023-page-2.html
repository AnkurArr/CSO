7) Hexadecimal number A equals —

✔ 10 (decimal)

Hex system: A = 10, B = 11, C = 12 … তাই A → 10.

8) Which memory loses data when power is switched off?

✔ RAM (volatile memory)

RAM = temporary storage → power off হলে সব মুছে যায়।

9) Which bus carries actual data?

✔ Data bus

Data bus দিয়ে CPU ↔ memory / I/O এর মধ্যে data চলাচল করে।

10) The device used to convert analog signal to digital is —

✔ ADC

ADC ইনপুট analog signal কে sampling + quantizing করে binary এ দেয়।

11) Which register stores the instruction currently being executed?

✔ Instruction Register (IR)

Fetch হওয়ার পর instruction আগে IR-এ যায় → তারপর CPU execute করে।

12) Which of the following is NOT an input device?

✔ Printer

Printer output দেয় — keyboard/mouse/scanner হলো input device।
