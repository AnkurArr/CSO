19) Which of the following is an output device?

✔ Monitor

Output devices display or produce the result — monitor, printer, speakers etc.

20) Which register stores data that is being transferred to or from memory?

✔ Memory Buffer Register (MBR)

MBR temporarily holds data during read/write operations with memory.

21) Binary number 1111 equals —

✔ 15 (decimal)

1×8 + 1×4 + 1×2 + 1×1 = 15.

22) The device used to store huge amount of data permanently is —

✔ Hard disk

Hard disks/SSDs are secondary storage — permanent and large capacity.

23) Which unit is used to measure clock speed?

✔ Hertz (Hz)

Clock frequency is measured in Hz (cycles per second) — MHz/GHz for CPUs.

24) The binary equivalent of decimal number 5 is —

✔ 101

5 = 4 + 1 → binary representation is 101.
