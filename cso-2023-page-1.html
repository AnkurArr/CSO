1) Which of the following is the fastest memory?

✔ Cache memory

Cache CPU-এর কাছে থাকে — তাই access time সবচেয়ে কম।

2) The register that holds the address of the next instruction is —

✔ Program Counter (PC)

PC সবসময় next instruction-এর address ধরে।

3) A group of 4 bits is called —

✔ Nibble

1 nibble = 4 bits, 1 byte = 8 bits.

4) Binary number 1010 equals —

✔ 10 (decimal)

1×8 + 0×4 + 1×2 + 0×1 = 10.

5) Which circuit performs arithmetic operations?

✔ ALU

ALU → arithmetic + logic কাজ করে।

6) The binary equivalent of octal number 7 is —

✔ 111

Octal digit 7 → binary 111.
