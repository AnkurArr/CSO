43) Which of the following is an input device?

✔ Scanner

A scanner captures images/text and sends them to the computer — input.

44) Decimal number 9 in binary is —

✔ 1001

9 = 8 + 1 → binary representation = 1001.

45) Which memory is the closest to the CPU?

✔ Cache memory

Cache sits between CPU and RAM and stores frequently used data.

46) The full form of ROM is —

✔ Read Only Memory

ROM stores permanent instructions needed during startup (booting).

47) Which of the following stores data permanently?

✔ Secondary storage

Hard disk / SSD keeps data even when power is OFF — permanent storage.

48) The binary equivalent of decimal number 3 is —

✔ 11

3 = 2 + 1 → binary = 11.
