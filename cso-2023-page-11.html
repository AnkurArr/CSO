61) Which device is used to record sound?

✔ Microphone

A microphone captures audio and sends it to the computer — input device.

62) Decimal number 7 in binary is —

✔ 111

7 = 4 + 2 + 1 → binary representation is 111.

63) Which unit performs comparison operations?

✔ ALU

ALU not only calculates — it also compares values (>, <, =).

64) The binary equivalent of decimal number 12 is —

✔ 1100

12 = 8 + 4 → binary = 1100.

65) Which storage is cheapest but slowest?

✔ Secondary storage

Hard disks/SSDs provide large, cheap storage — but slower than RAM/cache.

66) Which of the following is an example of primary memory?

✔ ROM

Primary memory includes RAM and ROM — directly accessible by CPU.
