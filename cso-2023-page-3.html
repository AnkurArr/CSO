13) Which component controls all activities of the computer?

✔ Control Unit (CU)

The Control Unit directs and coordinates all operations inside the CPU.

14) Which number system uses base 2?

✔ Binary

Binary system uses only two digits: 0 and 1 (base = 2).

15) 1 KB equals —

✔ 1024 bytes

1 KB = 210 bytes = 1024 bytes.

16) Which component performs logical operations?

✔ ALU

The Arithmetic Logic Unit performs arithmetic and logical operations.

17) The full form of CPU is —

✔ Central Processing Unit

CPU is the main processing unit of the computer system.

18) Which memory is non-volatile?

✔ ROM

ROM keeps data even when power is switched off.
