37) Which memory stores BIOS?

✔ ROM

BIOS firmware must stay even when power is OFF — stored in ROM.

38) Which of the following is NOT a primary memory?

✔ Hard disk

Hard disk is secondary storage — not directly accessed by CPU.

39) Decimal number 2 in binary is —

✔ 10

2 = 1×2 + 0×1 → binary = 10.

40) The register that stores the address of the instruction being fetched is —

✔ Program Counter

PC always points to the address of the next instruction to fetch.

41) Which device is used to enter text?

✔ Keyboard

Keyboard is the standard text-input device.

42) Which number system uses base 10?

✔ Decimal

Decimal system has digits 0–9 and base = 10.
