23) ______ are used to store a single binary digit.

✔ Flip-flops

একটা flip-flop = 1 bit data store করে।

24) ______ is the smallest unit of information in a computer.

✔ Bit

Bit = 0 বা 1.

25) The arithmetic unit of a computer is called ______.

✔ ALU (Arithmetic Logic Unit)

ALU → addition, subtraction, logic ইত্যাদি।

26) A group of 8 bits together makes one ______.

✔ Byte

1 byte = 8 bits.

27) ______ is a register that keeps the address of the next instruction.

✔ Program Counter (PC)

PC সবসময় next instruction-এর address ধরে।

28) ______ is used to convert assembly language to machine language.

✔ Assembler

Assembler → mnemonic → binary code.
