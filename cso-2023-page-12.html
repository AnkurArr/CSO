67) Which device is used to store data permanently?

✔ Hard disk / SSD

Secondary storage stores data permanently even when power is off.

68) Binary number 100 equals —

✔ 4 (decimal)

1×4 + 0×2 + 0×1 = 4.

69) Which unit fetches instructions from memory?

✔ Control Unit

CU fetches, decodes and controls execution of instructions.

70) Which memory is faster than RAM but smaller?

✔ Cache memory

Cache is small, very fast memory used to speed up data access.

71) Which of the following is NOT a storage device?

✔ Mouse

Mouse is an input device. It cannot store data.

72) Decimal number 1 in binary is —

✔ 1

1 remains 1 in binary.
