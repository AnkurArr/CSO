Part-1 MCQ

1) Which memory is the fastest?

✔ Cache memory

Cache sits nearest to the CPU and has the lowest access time.

2) The brain of the computer is —

✔ CPU

CPU controls, processes and executes all instructions.

3) Binary number 10 equals —

✔ 2 (decimal)

Binary 10 = (1×2) + (0×1) = 2.

4) Which unit performs arithmetic calculations?

✔ ALU

ALU performs arithmetic and logical operations.

5) Which device stores data permanently?

✔ Hard disk / SSD

Secondary storage retains data even when power is OFF.

6) ROM is —

✔ Non-volatile

ROM keeps data permanently.

7) Which device is used to enter text?

✔ Keyboard

Keyboard is the primary text-input device.

8) Decimal number 4 in binary is —

✔ 100

4 = 1×4 + 0×2 + 0×1 → 100.

9) Which component coordinates all parts of the computer?

✔ Control Unit

The Control Unit sends control signals and manages execution.

10) Which of the following is an output device?

✔ Monitor

Monitor displays processed information to the user.

Part-2 MCQ

11) Binary number 101 equals —

✔ 5 (decimal)

1×4 + 0×2 + 1×1 = 5.

12) Which bus carries data?

✔ Data bus

Data bus transfers actual data between CPU and memory/I-O.

13) Primary memory is —

✔ RAM

Primary memory is directly accessed by CPU.

14) Decimal 9 in binary is —

✔ 1001

9 = 8 + 1 → 1001.

15) Which device prints output on paper?

✔ Printer

Printer produces hard-copy output.

16) BIOS is stored in —

✔ ROM

BIOS must stay after power off, so it is stored in ROM.

17) Which memory is smaller and faster than RAM?

✔ Cache memory

Cache is small but extremely fast.

18) Decimal 3 in binary is —

✔ 11

3 = 2 + 1 → 11.

19) Which register stores next instruction address?

✔ Program Counter

PC always points to the next instruction.

20) Which memory is also called auxiliary memory?

✔ Secondary memory

Secondary memory is used for long-term storage.

Part-3 MCQ (End)

21) Which device records sound?

✔ Microphone

Microphone captures audio — input device.

22) Decimal 8 in binary is —

✔ 1000

8 = 1×8 + 0×4 + 0×2 + 0×1.

23) Which part executes instructions?

✔ CPU

CPU fetches, decodes and executes instructions.

24) Which device is used for backup?

✔ External storage (USB / external HDD)

Backups are stored on removable secondary storage.

25) Decimal 6 in binary is —

✔ 110

6 = 4 + 2 → 110.

26) Which component converts digital to sound?

✔ Speaker

Speakers output sound from digital signals.

27) Which unit controls I/O operations?

✔ I/O Unit

The I/O unit manages communication with devices.

28) Decimal 7 in binary is —

✔ 111

7 = 4 + 2 + 1 → 111.

29) Which memory loses data when power goes off?

✔ RAM

RAM is volatile memory.

30) Which bus carries control signals?

✔ Control bus

Control bus sends timing and control signals.
